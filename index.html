Treinamento Git/GitHub para Iniciantes

Hello, world! A nossa jornada Git começa aqui!

O que é Git?

Git é um sistema de controle de versão que permite acompanhar todas as mudanças feitas em arquivos de um projeto. Ele registra cada alteração, possibilita voltar no tempo, criar ramificações (branches) para testar ideias sem quebrar o projeto principal e facilita o trabalho em equipe, evitando perda de código.

O que é GitHub?

GitHub é uma plataforma online que hospeda repositórios Git, permitindo colaboração entre desenvolvedores. Ele oferece ferramentas para revisão de código, gerenciamento de projetos e integração contínua, facilitando o desenvolvimento colaborativo e a publicação de projetos.
